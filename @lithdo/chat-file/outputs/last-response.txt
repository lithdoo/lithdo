# 绿手指

老陈的手指是绿色的。

不是沾了油漆，也不是生了怪病。那绿是活的，像初春柳芽尖上的嫩，又像深潭青苔的润。他能让枯木发芽，能让濒死的吊兰一夜之间垂下翡翠瀑布。

邻居们都叫他“绿手指”，半是羡慕半是好奇。老陈只是笑，每天清晨提着小喷壶，在阳台上侍弄他的植物王国。他的阳台是整栋灰白楼房唯一的绿洲，爬山虎从七楼垂到三楼，三角梅开得泼辣，薄荷与罗勒的香气在夏夜里流淌。

没人知道这能力的来历。老陈自己也不太明白——三年前那场大病后，他醒来就发现指尖泛着淡淡的绿。起初他吓坏了，直到无意中触到窗台那盆奄奄一息的茉莉，枯枝竟以肉眼可见的速度抽出新芽。

从此他成了植物的医生。

但老陈有个秘密：每救活一株植物，他手指的绿色就会淡一分。起初只是不易察觉的变化，直到上个月救活那棵百年老榕树——那是开发商要砍掉的，他深夜偷偷去抚摸树干，第二天老榕树发出新枝，而他的中指几乎看不出绿色了。

“还剩三根。”他数着左手仅存的绿手指，像在数生命的倒计时。

今天来了不速之客。穿西装的男人递上名片：“陈先生，我们老板愿出五十万，请您去救他的私人收藏——一株濒危的龙血树。”

老陈摇头：“我治不了。”

“一百万。”

“不是钱的问题。”他握紧左手，藏进口袋。

西装男离开时在门口停住：“那棵树是从热带雨林运来的，为了它，砍了半亩森林。”他顿了顿，“老板说，救不活就烧了当柴火。”

门关上了。老陈站在阳台上，看着自己的手。夕阳下，无名指的绿莹莹发光。

深夜，他站在那株龙血树前。它确实快死了，叶片枯黄卷曲，树干渗出暗红的树脂像眼泪。老陈伸出手，又缩回。他想起了自己的茉莉，想起了邻居阿婆的兰花，想起了幼儿园孩子们种下的向日葵——都是这些绿色手指救活的。

“如果这是最后一件……”他喃喃道，将手掌贴上树干。

绿光从指尖流淌而出，像小溪渗入干裂的土地。龙血树的叶片开始舒展，枯黄褪去，新芽从顶端冒出。老陈看着自己的手：无名指的绿色正迅速消退，接着是小指……

当龙血树完全恢复生机时，他左手的绿色全部消失了。手掌苍白如纸，只有常年劳作留下的茧。

西装男递来支票。老陈没接，只是问：“你会照顾好它吗？”

“当然，这可是价值连城的……”

“它不只是收藏品。”老陈转身离开，“它活过，这就够了。”

回家的路上，他习惯性地摸向口袋里的种子——那是他随身带的，随时准备给需要的人。掏出来时，他愣住了：掌心的种子在月光下闪着微弱的绿光。

不是来自手指。

是来自他的掌心，来自更深处的地方。

老陈忽然明白了。绿色从未真正住在指尖，它只是路过那里，像河流经过山谷。真正的源头，在他每一次选择让生命延续的瞬间。

他笑了，对着空荡荡的手掌。阳台上的植物在夜风中沙沙作响，仿佛在说：我们一直都在。

第二天，邻居们惊讶地发现，老陈的阳台又添了新绿——不是来自他的手指，而是来自他分给每个人的种子，和那双依然温柔照料生命的手。

绿手指消失了。

但绿色，才刚刚开始生长。
